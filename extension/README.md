# 源泉カンタン計算（Chrome拡張機能） ご利用ガイド

このツールは、原稿料や講演料などの源泉徴収税額をGoogle Chrome上で素早く計算し、計算結果をコピーできる専用ツールです。
開発の知識がない方でも、以下の手順に沿って簡単に使い始めることができます。

---

## 1. 初期設定（インストールの手順）

最初だけ、Google Chromeにこの計算機を追加する作業が必要です。所要時間は1〜2分です。

**手順:**
1. お使いの **Google Chrome** ブラウザを開きます。
2. 画面上部のアドレスバー（URLが入っているところ）に半角で `chrome://extensions/` と入力して、Enterキーを押します。
   （※または、右上の「︙」メニュー ＞「拡張機能」＞「拡張機能を管理」をクリックしても開けます）
3. 画面の **右上** にある **「デベロッパー モード」** というスイッチを探し、クリックして **オン（色がつく状態）** にします。
4. 画面の **左上** に新しく表示された **「パッケージ化されていない拡張機能を読み込む」** というボタンをクリックします。
5. フォルダを選択する画面が開くので、このツールが保存されている `extension` フォルダ（例: `C:\Users\hayas\Documents\Gensenkeisan\extension`）を選択し、右下の「フォルダの選択」をクリックします。
6. 画面に「源泉カンタン計算」というパネルが表示されれば、インストールは完了です！

### 💡 おすすめの設定（便利に使うために）
Chromeの画面右上にある **「ジグソーパズルのマーク（拡張機能アイコン）」** をクリックします。
リストの中から「源泉カンタン計算」を探し、隣にある **「ピンのマーク（📌）」** をクリックして青色にします。
これを行うと、Chromeの右上に常にアイコンが表示され、いつでもワンクリックで計算機を開けるようになります。

---

## 2. 使い方

1. Chromeの右上に追加されたアイコンをクリックします。
2. 小さな計算機（ポップアップ）が開きます。
3. **「請求額（円）」** の欄に、数字で金額を入力します。（例: 50000）
   - ※入力と同時に、全自動で下の金額が計算されます。
4. 請求書の記載に合わせて、**「消費税」** のスイッチ（税別 または 税込）を選びます。
   - **税別**：入力した金額に対して10%の消費税が加算され、そこから源泉税が引かれます。
   - **税込**：入力した金額を「総額」として扱い、そこから源泉税が引かれます（消費税は込みとして計算します）。
5. 計算が終わったら、各金額の右横にある **「📋（コピーボタン）」** をクリックすると、その金額の数字だけをコピーできます。そのままExcelや会計ソフトに「貼り付け（ペースト）」してください。（※金額のカンマなどは除かれた純粋な数字がコピーされるので、エラーになりません）
6. 一番下の **「結果をすべてコピー」** ボタンを押すと、請求額・消費税・源泉税・手取り額の内訳テキストをまるごとコピーできます。メールや社内チャットでの連絡時に便利です。

---

## 3. 自動計算の仕様について
- **対象**: 原稿料、講演料などの報酬（10.21% および 20.42%の税率）
- **判定**: 支払い金額が100万円以下の部分は「金額 × 10.21%」、100万円を超える部分は税率が上がり「20.42%」として、国税庁の標準的な計算式を自動で適用します。
- **端数処理**: 消費税引き上げ分や源泉徴収税額に小数点が出た場合は、一般的な実務に合わせて「切り捨て」で計算しています。
